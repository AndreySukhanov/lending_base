# Быстрый старт - Система Сценариев

## 1. Настройка Backend

### Проверить OpenAI API key

Убедитесь, что в `backend/.env` есть:
```bash
OPENAI_API_KEY=ваш-ключ-openai
```

**Примечание:** Генераторы используют тот же OpenAI ключ, дополнительных ключей не требуется.

### Инициализировать БД

```bash
cd backend
python init_db.py
```

### Запустить backend

```bash
cd backend
uvicorn app.main:app --reload
```

## 2. Запуск Frontend

```bash
cd frontend
npm run dev
```

## 3. Использование

### Генерация со сценарием
1. Откройте http://localhost:3000/generate
2. Выберите сценарий из списка
3. Заполните остальные поля
4. Нажмите "Сгенерировать копирайт"

### Генераторы имён и отзывов
1. Откройте http://localhost:3000/generators
2. Используйте два генератора для создания имён и отзывов

### Управление сценариями
1. На странице генерации нажмите "Управление" рядом с выбором сценария
2. Создавайте, редактируйте или удаляйте сценарии

## Готовые сценарии

1. **Статья-расследование** - журналистское расследование
2. **Интервью на ток-шоу** - скандальное интервью
3. **Отменённое интервью** - "запрещённый" контент
4. **Интервью изданию** - классическое интервью
